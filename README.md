# HyperTrophy 🔱

**The Elite Calisthenics Progression Engine.** 

HyperTrophy is a high-performance, offline-first workout tracker designed for bodyweight athletes who prioritize progression over fluff. No social feeds, no ads, just pure data-driven hypertrophy.

![HyperTrophy UI Preview](https://via.placeholder.com/1200x600/121212/D4FF00?text=🔱+HYPERTROPHY+ATHLETE+DASHBOARD)

## ⚡ Core Features

- **Skill Tree Progression**: Navigate a predefined roadmap of Bodyweight exercises from Knee Pushups to Full Planche.
- **Dynamic Skill Unlock**: The app automatically detects when you've hit your volume thresholds and unlocks the next tier of difficulty.
- **Chaos Mode™**: Stuck in a plateau? Generate a randomized, equipment-aware workout to shock your CNS.
- **Superset Engine**: Seamlessly toggle between paired exercises with a dedicated low-rest timer.
- **Data Portability**: Full Base64 state export/import. Your data belongs to you, always.
- **Volume Heatmaps**: Track your muscle group balance (Push/Pull/Core/Legs) over time to avoid imbalances.

## 🛠️ Technology Stack

- **Framework**: Expo / React Native
- **Router**: Expo Router (File-based)
- **State Management**: Zustand (with Persist)
- **Storage**: AsyncStorage (Expo Go Compatible)
- **Haptics**: Optimized tactile feedback for log events
- **Animations**: Reanimated & Expo Linear Gradient

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- Expo Go app on your smartphone

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aadel/hypertrophy.git
   cd hypertrophy
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```

4. **Scan the QR code** in your terminal using the Expo Go app.

## 🏋️ Curated Routines

HyperTrophy comes pre-loaded with four baseline routines:
1. **Push Basics**: Focused on horizontal and vertical pressing foundations.
2. **Pull Basics**: Building your first pullup and ring-row volume.
3. **Core Crusher**: Fundamental hollow-body and L-sit progressions.
4. **Leg Day**: Unilateral leg strength with Bulgarian Split Squats.

## 🔐 Privacy & Security

HyperTrophy is **100% Offline-Only**. We do not collect, store, or sell any of your workout data. All persistence happens locally on your device's secure storage.
